# ПОЛНАЯ-ХУЙНЯ-МЕТР
Версия 1.2

## Схема подключения (общая)
1. **2-позиционный переключатель (SPDT, 3 контакта):**
   - Общий контакт (COM) → **GND**
   - Контакт положения 1 → **D3**
   - Контакт положения 2 → **D4**

2. **Кнопка:**
   - Один вывод кнопки → **D5**
   - Второй вывод → **GND**

3. **Потенциометр (10 кОм):**
   - Один крайний вывод → **5V**
   - Другой крайний вывод → **GND**
   - Средний вывод (ползунок) → **A2**

4. **Пищалка (пассивная):**
   - `+` → **D6**
   - `-` → **GND**

5. **Светодиод:**
   - Используется встроенный LED Arduino Nano на **D13**.

## Вариант со сервоприводом
Файл: `nano_servo_buzzer_switch.ino`

- Servo signal → **D9**
- Servo VCC → **5V**
- Servo GND → **GND**

## Вариант с TM1637 (4x7-сегм. индикатор)
Файл: `nano_tm1637_buzzer_switch.ino`

- TM1637 CLK → **D8**
- TM1637 DIO → **D9**
- TM1637 VCC → **5V**
- TM1637 GND → **GND**

Зависимость (Arduino Library Manager): `TM1637Display`.
